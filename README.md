# StudyPlusAI

StudyPlusAI is an intelligent student learning and progress tracking platform designed to help students manage their studies effectively, monitor academic performance, and receive personalized learning support.

## Features

* Student progress tracking
* Learning roadmap management
* Performance analytics dashboard
* Course and subject management
* Goal setting and tracking
* User authentication and authorization
* Responsive and user-friendly interface
* Future AI-powered learning recommendations

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* JavaScript

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

## Project Structure

```text
studyplus-ai/
├── frontend/

```

## Installation

### Clone the Repository

```bash
git clone https://github.com/dharani232007/studyplus-ai.git
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
npm run dev
```

### Install Backend Dependencies

```bash
cd backend
npm install
npm start
```
